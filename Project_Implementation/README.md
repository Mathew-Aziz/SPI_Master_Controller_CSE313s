- `final_project_student/harness/`
- `final_project_student/harness/examples/sv_only/`
- The provided project specification and grading interface requirements.

## Mapping to harness starter

- Keep using the official harness files without modifying DUT RTL:
  - `harness/apb_if.sv`
  - `harness/spi_if.sv`
  - `harness/dut_wrapper.sv`
- Use `harness/examples/sv_only/` as the baseline style for SV-only flow:
  - `assertions/`, `env/`, `sequences/`, `tb/`, `tests/`

## Notes

- This is a hierarchy plan and organization guide.
- No SystemVerilog source content is modified by this step.
- The Makefile in this folder should implement the mandatory grading targets:
  - `compile`, `run`, `regress`, `run_bonus`, `cov`, `clean`
